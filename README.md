# Annotation tool - Инструмент для лингвистической разметки текстов
 — это инструмент для автоматической и ручной лингвистической разметки текстов на русском языке.

### Основные возможности
Автоматическая разметка с использованием современных NLP-библиотек (Stanza, spaCy)

Хранение и управление текстовыми данными и результатами разметки

Экспорт результатов в различные форматы (CoNLL-U, JSON)

Анализ и визуализация лингвистических особенностей текстов
